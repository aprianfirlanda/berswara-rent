# User Flow: Berswara Rent

## 1. Discovery Flow
1. User arrives on **Home Page**.
2. User scrolls to see **Featured Products** or clicks **Catalog**.
3. User filters/browses by category (e.g., "Strollers").

## 2. Selection & Information Flow
1. User clicks on a specific product (e.g., "Bugaboo Butterfly").
2. User reviews **Product Details** (Specs, Pricing, Availability).
3. User checks **How to Rent** to understand the process.

## 3. Booking Inquiry Flow (Static MVP)
1. User clicks **"Rent via WhatsApp"** or **"Inquire Availability"** button on Product Page.
2. User is redirected to WhatsApp with a pre-filled message:
   *"Hi Berswara Rent, I'm interested in renting the [Product Name]. Is it available for [Dates]?"*
3. Manual communication follows offline.

## 4. Post-Inquiry (Offline/Manual)
1. Admin confirms availability.
2. User provides ID/Deposit.
3. Delivery/Pick-up arranged.
